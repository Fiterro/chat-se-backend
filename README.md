# chat-se-backend

## Description

self education backend project

## Project setup
Need to create copy of `config.bak` folder as `config` and edit configurations

## Installation

```bash
- $ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test (not supported yet)

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
